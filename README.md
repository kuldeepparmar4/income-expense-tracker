#  FinTrack — Income & Expense Tracker

A full-stack financial management dashboard built with the MERN stack. Users can log transactions, visualize spending trends, and track their financial health in real time.


##  Live Demo

https://income-expense-tracker-app-six.vercel.app

> **Note:** Backend is hosted on Render free tier. First request may take 30-60 seconds to wake up.

---

##  Features

-  Add income and expense transactions
-  View total balance, income, and expenses in real time
-  Filter transactions by type (income/expense)
-  Search transactions by title
-  Delete transactions
-  Pie chart — expenses by category
-  Bar chart — monthly income vs expenses
-  Fully responsive design
-  RESTful API with full CRUD operations

---

##  Tech Stack

### Frontend
| Technology | Purpose |
|---|---|
| React.js | UI library for building components |
| Chart.js | Data visualization (pie and bar charts) |
| Axios | HTTP requests to backend API |
| React Icons | Icon library |
| CSS3 | Styling and responsive design |

### Backend
| Technology | Purpose |
|---|---|
| Node.js | JavaScript runtime environment |
| Express.js | Web framework for building REST APIs |
| MongoDB | NoSQL database for storing transactions |
| Mongoose | MongoDB object modeling for Node.js |
| dotenv | Environment variable management |
| CORS | Cross-origin resource sharing |
| Nodemon | Auto-restart server during development |

### Deployment
| Service | Purpose |
|---|---|
| Vercel | Frontend deployment |
| Render | Backend deployment |
| MongoDB Atlas | Cloud database |
